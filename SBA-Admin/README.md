# Student Results Manager

A comprehensive web application for managing student results, grades, and reports.

## Features

- **Multiple Datasets**: Work with multiple independent result datasets
- **Dynamic Tables**: Create and manage class/subject tables
- **Flexible CAT Columns**: Customize assessment columns with configurable maximum scores
- **Smart Mark Input**: Enter marks in fractions (45/50), percentages (90%), or direct numbers
- **Automatic Calculations**: CAT totals, overall scores, and positions calculated automatically
- **Statistics View**: Overview of all students across tables
- **Terminal Reports**: Generate detailed student report cards
- **Bulk Operations**: Import/export multiple datasets
- **Print Support**: Generate A4-sized printable reports
- **Data Integrity Checks**: Validate and repair data structure
